# Device Settings Audio Port — Specifications

| Req ID | Test Scope |
|:-------|:--------------------------------------|
| `VTS‑DSAUDIO‑001` | SHALL initialize the Device Settings Audio Port sub-system via `dsAudioPortInit()` returning `dsERR_NONE` and SHALL terminate it via `dsAudioPortTerm()` returning `dsERR_NONE`. The module SHALL support re-initialization after a prior termination without error. |
| `VTS‑DSAUDIO‑002` | SHALL return a valid, non-null audio port handle for each supported port type and index via `dsGetAudioPort()` returning `dsERR_NONE`, with repeated calls for the same port type and index returning an equal handle. |
| `VTS‑DSAUDIO‑003` | SHALL retrieve the current audio stream format of a valid audio port via `dsGetAudioFormat()` returning `dsERR_NONE` and a valid `dsAudioFormat_t` value. |
| `VTS‑DSAUDIO‑004` | SHALL enable and disable each supported audio port via `dsEnableAudioPort()` returning `dsERR_NONE`, and the enable state retrieved via `dsIsAudioPortEnabled()` SHALL match the state that was set. |
| `VTS‑DSAUDIO‑005` | SHALL set the audio compression level of a supported audio port via `dsSetAudioCompression()` returning `dsERR_NONE`, and the value retrieved via `dsGetAudioCompression()` SHALL match the level that was set. |
| `VTS‑DSAUDIO‑006` | SHALL set the dialog enhancement level of a supported audio port via `dsSetDialogEnhancement()` returning `dsERR_NONE`, and the value retrieved via `dsGetDialogEnhancement()` SHALL match the level that was set. If dialog enhancement is not supported by the DUT, `dsSetDialogEnhancement()` and `dsGetDialogEnhancement()` SHALL return `dsERR_OPERATION_NOT_SUPPORTED`. |
| `VTS‑DSAUDIO‑007` | SHALL set the Dolby volume mode of a supported audio port via `dsSetDolbyVolumeMode()` returning `dsERR_NONE`, and the mode retrieved via `dsGetDolbyVolumeMode()` SHALL match the value that was set. If Dolby volume mode is not supported by the DUT, `dsSetDolbyVolumeMode()` and `dsGetDolbyVolumeMode()` SHALL return `dsERR_OPERATION_NOT_SUPPORTED`. |
| `VTS‑DSAUDIO‑008` | SHALL set the intelligent equalizer mode of a supported audio port via `dsSetIntelligentEqualizerMode()` returning `dsERR_NONE`, and the mode retrieved via `dsGetIntelligentEqualizerMode()` SHALL match the value that was set. If the intelligent equalizer is not supported by the DUT, `dsSetIntelligentEqualizerMode()` and `dsGetIntelligentEqualizerMode()` SHALL return `dsERR_OPERATION_NOT_SUPPORTED`. |
| `VTS‑DSAUDIO‑009` | SHALL set the volume leveller mode and level of a supported audio port via `dsSetVolumeLeveller()` returning `dsERR_NONE`, and the settings retrieved via `dsGetVolumeLeveller()` SHALL match the values that were set. If the volume leveller is not supported by the DUT, `dsSetVolumeLeveller()` and `dsGetVolumeLeveller()` SHALL return `dsERR_OPERATION_NOT_SUPPORTED`. |
| `VTS‑DSAUDIO‑010` | SHALL set the bass enhancer boost level of a supported audio port via `dsSetBassEnhancer()` returning `dsERR_NONE`, and the level retrieved via `dsGetBassEnhancer()` SHALL match the value that was set. If the bass enhancer is not supported by the DUT, `dsSetBassEnhancer()` and `dsGetBassEnhancer()` SHALL return `dsERR_OPERATION_NOT_SUPPORTED`. |
| `VTS‑DSAUDIO‑011` | SHALL enable and disable the surround decoder of a supported audio port via `dsEnableSurroundDecoder()` returning `dsERR_NONE`, and the enable state retrieved via `dsIsSurroundDecoderEnabled()` SHALL match the state that was set. If the surround decoder is not supported by the DUT, `dsEnableSurroundDecoder()` and `dsIsSurroundDecoderEnabled()` SHALL return `dsERR_OPERATION_NOT_SUPPORTED`. |
| `VTS‑DSAUDIO‑012` | SHALL set the DRC (Dynamic Range Control) mode of a supported audio port via `dsSetDRCMode()` returning `dsERR_NONE`, and the mode retrieved via `dsGetDRCMode()` SHALL match the value that was set. If DRC is not supported by the DUT, `dsSetDRCMode()` and `dsGetDRCMode()` SHALL return `dsERR_OPERATION_NOT_SUPPORTED`. |
| `VTS‑DSAUDIO‑013` | SHALL set the surround virtualizer mode and boost of a supported audio port via `dsSetSurroundVirtualizer()` returning `dsERR_NONE`, and the settings retrieved via `dsGetSurroundVirtualizer()` SHALL match the values that were set. If the surround virtualizer is not supported by the DUT, `dsSetSurroundVirtualizer()` and `dsGetSurroundVirtualizer()` SHALL return `dsERR_OPERATION_NOT_SUPPORTED`. |
| `VTS‑DSAUDIO‑014` | SHALL set the Media Intelligent (MI) steering enable state of a supported audio port via `dsSetMISteering()` returning `dsERR_NONE`, and the state retrieved via `dsGetMISteering()` SHALL match the value that was set. If MI steering is not supported by the DUT, `dsSetMISteering()` and `dsGetMISteering()` SHALL return `dsERR_OPERATION_NOT_SUPPORTED`. |
| `VTS‑DSAUDIO‑015` | SHALL set the graphic equalizer mode of a supported audio port via `dsSetGraphicEqualizerMode()` returning `dsERR_NONE`, and the mode retrieved via `dsGetGraphicEqualizerMode()` SHALL match the value that was set. If the graphic equalizer is not supported by the DUT, `dsSetGraphicEqualizerMode()` and `dsGetGraphicEqualizerMode()` SHALL return `dsERR_OPERATION_NOT_SUPPORTED`. |
| `VTS‑DSAUDIO‑016` | SHALL enable and disable the loudness equivalence (LE) configuration of a supported audio port via `dsEnableLEConfig()` returning `dsERR_NONE`, and the configuration retrieved via `dsGetLEConfig()` SHALL match the state that was set. If LE configuration is not supported by the DUT, `dsEnableLEConfig()` and `dsGetLEConfig()` SHALL return `dsERR_OPERATION_NOT_SUPPORTED`. |
| `VTS‑DSAUDIO‑017` | SHALL retrieve the list of supported MS12 audio profiles via `dsGetMS12AudioProfileList()` returning `dsERR_NONE`, retrieve the currently active MS12 audio profile via `dsGetMS12AudioProfile()` returning `dsERR_NONE`, and set an MS12 audio profile via `dsSetMS12AudioProfile()` returning `dsERR_NONE` for valid profile names on supported ports. If MS12 audio profiles are not supported by the DUT, `dsGetMS12AudioProfileList()`, `dsGetMS12AudioProfile()`, and `dsSetMS12AudioProfile()` SHALL return `dsERR_OPERATION_NOT_SUPPORTED`. |
| `VTS‑DSAUDIO‑018` | SHALL set the stereo mode of a supported audio port via `dsSetStereoMode()` returning `dsERR_NONE`, and the mode retrieved via `dsGetStereoMode()` SHALL match the value that was set. |
| `VTS‑DSAUDIO‑019` | SHALL set the stereo auto mode of a supported audio port via `dsSetStereoAuto()` returning `dsERR_NONE`, and the mode retrieved via `dsGetStereoAuto()` SHALL match the value that was set. If stereo auto mode is not supported by the DUT, `dsSetStereoAuto()` and `dsGetStereoAuto()` SHALL return `dsERR_OPERATION_NOT_SUPPORTED`. |
| `VTS‑DSAUDIO‑020` | SHALL set the audio gain of a supported audio port via `dsSetAudioGain()` returning `dsERR_NONE`, and the gain retrieved via `dsGetAudioGain()` SHALL match the value that was set. If audio gain control is not supported by the DUT, `dsSetAudioGain()` and `dsGetAudioGain()` SHALL return `dsERR_OPERATION_NOT_SUPPORTED`. |
| `VTS‑DSAUDIO‑021` | SHALL set the audio volume level of a supported audio port via `dsSetAudioLevel()` returning `dsERR_NONE`, and the level retrieved via `dsGetAudioLevel()` SHALL match the value that was set. If audio volume level control is not supported by the DUT, `dsSetAudioLevel()` and `dsGetAudioLevel()` SHALL return `dsERR_OPERATION_NOT_SUPPORTED`. |
| `VTS‑DSAUDIO‑022` | SHALL set the audio delay of a supported audio port via `dsSetAudioDelay()` returning `dsERR_NONE`, and the delay retrieved via `dsGetAudioDelay()` SHALL match the value that was set. |
| `VTS‑DSAUDIO‑023` | SHALL mute and unmute the audio output of a supported audio port via `dsSetAudioMute()` returning `dsERR_NONE`, and the mute state retrieved via `dsIsAudioMute()` SHALL match the state that was set. |
| `VTS‑DSAUDIO‑024` | SHALL configure the Dolby ATMOS output mode of a supported audio port via `dsSetAudioAtmosOutputMode()` returning `dsERR_NONE`, and SHALL retrieve the ATMOS capability of the connected sink device via `dsGetSinkDeviceAtmosCapability()` returning `dsERR_NONE` and a valid `dsATMOSCapability_t` value. |
| `VTS‑DSAUDIO‑025` | SHALL enable and disable associated audio mixing via `dsSetAssociatedAudioMixing()` returning `dsERR_NONE`, and the mixing state retrieved via `dsGetAssociatedAudioMixing()` SHALL match the state that was set. |
| `VTS‑DSAUDIO‑026` | SHALL set the fader control mixer balance in the range [-32, +32] via `dsSetFaderControl()` returning `dsERR_NONE`, and the balance retrieved via `dsGetFaderControl()` SHALL match the value that was set. |
| `VTS‑DSAUDIO‑027` | SHALL configure the audio input mixer levels for each supported audio input type via `dsSetAudioMixerLevels()` returning `dsERR_NONE` for valid volume values. If audio mixer level control is not supported by the DUT, `dsSetAudioMixerLevels()` SHALL return `dsERR_OPERATION_NOT_SUPPORTED`. |
| `VTS‑DSAUDIO‑028` | SHALL set the AC4 primary language code via `dsSetPrimaryLanguage()` and the AC4 secondary language code via `dsSetSecondaryLanguage()`, each returning `dsERR_NONE` for valid 3-letter language codes, and the codes retrieved via `dsGetPrimaryLanguage()` and `dsGetSecondaryLanguage()` SHALL match the codes that were set. |
| `VTS‑DSAUDIO‑029` | SHALL retrieve the bitwise-OR-ed audio capabilities of each audio port via `dsGetAudioCapabilities()` returning `dsERR_NONE` with the value matching the `AudioCapabilities` declared in the device profile, and the MS12 capabilities via `dsGetMS12Capabilities()` returning `dsERR_NONE` with the value matching the `MS12Capabilities` in the device profile. |
| `VTS‑DSAUDIO‑030` | SHALL report MS11 multistream decode support via `dsIsAudioMSDecode()` returning `dsERR_NONE` with the result matching the `MS11Decode` field in the device profile, and MS12 multistream decode support via `dsIsAudioMS12Decode()` returning `dsERR_NONE` with the result matching the `MS12Decode` field in the device profile. |
| `VTS‑DSAUDIO‑031` | SHALL retrieve the supported ARC types of the connected ARC/eARC device via `dsGetSupportedARCTypes()` returning `dsERR_NONE` for HDMI ARC-type ports, configure Short Audio Descriptor (SAD) data received from CEC via `dsAudioSetSAD()` returning `dsERR_NONE` for valid SAD lists, and enable and disable ARC/eARC audio routing via `dsAudioEnableARC()` returning `dsERR_NONE`. If ARC/eARC is not supported by the DUT, `dsGetSupportedARCTypes()`, `dsAudioSetSAD()`, and `dsAudioEnableARC()` SHALL return `dsERR_OPERATION_NOT_SUPPORTED`. |
| `VTS‑DSAUDIO‑032` | SHALL report the audio output port connection status via `dsAudioOutIsConnected()` returning `dsERR_NONE`, and SHALL successfully register event callbacks for audio output connection changes via `dsAudioOutRegisterConnectCB()`, audio format updates via `dsAudioFormatUpdateRegisterCB()`, and ATMOS capability changes via `dsAudioAtmosCapsChangeRegisterCB()`, each returning `dsERR_NONE`. If connection status reporting or the connection-change callback is not supported by the DUT, `dsAudioOutIsConnected()` and `dsAudioOutRegisterConnectCB()` SHALL return `dsERR_OPERATION_NOT_SUPPORTED`. |
| `VTS‑DSAUDIO‑033` | SHALL enforce the following error code contracts across all `dsAudio` APIs:<br>`dsAudioPortInit()` SHALL return `dsERR_ALREADY_INITIALIZED` when it is called while the module is already initialized<br>every `dsAudio` API SHALL return `dsERR_NOT_INITIALIZED` when it is invoked without prior initialization or after the module has already been terminated<br>every `dsAudio` API SHALL return `dsERR_INVALID_PARAM` when it is called with an invalid or null port handle, a NULL output pointer, or an out-of-range parameter value. |
