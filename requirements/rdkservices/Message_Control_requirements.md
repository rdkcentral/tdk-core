## Message_Control Plugin — Specifications


| Req ID | Test Scope |
|:-------|:--------------------------------------|
| `MC‑REQ‑001` | SHALL successfully get and toggle all trace levels for the Application module via the controls and enable APIs, and restore the original trace level after toggling |
| `MC‑REQ‑002` | SHALL successfully get and toggle all trace levels for the SysLog, OCDM, System, Monitor, DisplayInfo, DeviceInfo, PlayerInfo, and MessageControl plugin modules, and restore the original trace levels after toggling |
| `MC‑REQ‑003` | SHALL emit the statechange event with the correct plugin state when the MessageControl plugin is activated and deactivated |
